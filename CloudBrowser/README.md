CloudBrowser
============

it's a browser writen in qt4.7.4.

if you want release you need some dll like this:
/-CloudBrowser.exe
--codecs
       /--qcncodecs4.dll
--imageformats
        /--qgif4.dll
        /--qico4.dll
        /--qjpeg4.dll
        /--qmng4.dll
        /--qsvg4.dll
        /--qtiff4.dll
--libgcc_s_dw2-1.dll
--libssl32.dll
--mingwm10.dll
--phonon4.dll
--plugins
        /--NPSWF32_11_7_700_169.dll     to support flash swf 
--QtCore4.dll
--QtGui4.dll
--QtNetwork4.dll
--QtWebKit4.dll
--ssleay32.dll
